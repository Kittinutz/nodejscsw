# Welcome to Nodejs Server

## Purpose

> We develop for CSW 2/60

## How to install

```bash
git clone https://github.com/Kittinutz/nodejscsw.git
cd nodejscsw
npm install
npm run dev

# open web browser: http://127.0.0.1:3000
```
